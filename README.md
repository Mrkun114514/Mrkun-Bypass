# Mrkun Bypass 模组

一个 Minecraft Fabric 多版本兼容的乐子模组，模拟开挂功能提示但**不会真的有作弊效果**。

## 版本支持

| Minecraft 版本 | Fabric Loader | Fabric API |
|---------------|---------------|------------|
| 1.20.1 | 0.16.0+ | 0.90.0+ |
| 1.20.4 | 0.16.0+ | 0.97.0+ |
| 1.21 | 0.16.5+ | 0.104.0+ |
| 1.21.1 | 0.16.7+ | 0.107.0+ |
| 1.21.11 | 0.16.11+ | 0.111.0+ |

## 功能说明

### 快捷键列表
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 打开菜单 | **H** | 打开精美的功能菜单 |

| 无掉落伤害 (NoFall) | N | 模拟无掉落伤害开启提示 |
| 疾跑 (Sprint) | Z | 模拟疾跑开启提示 |
| 夜视 (Fullbright) | B | 模拟夜视开启提示 |

### 提示格式
按下对应按键后，聊天栏会显示：
- **白色文字**: `Mrkun Bypass：`
- **红色文字**: `功能名 已开启！` / `功能名 已关闭！`

### 菜单界面
按 **H** 键打开/关闭菜单，可以在菜单中点击按钮切换功能状态。
- 开启的功能显示绿色边框+绿色背景，关闭的显示红色边框+红色背景
- 菜单支持鼠标滚轮滚动，避免窗口太小时按钮被截断
- 底部显示会话统计（可在设置中关闭）

### 首次进入世界
首次佩戴模组进入世界时会弹出说明弹窗，并在左上角显示提示。

### 配置
配置文件保存在 `.minecraft/Mrkun_Bypass_config/mrkun_bypass_config.json`，包含：
- 各模块的开关状态
- 是否显示首次启动提示
- 是否显示左上角启动消息
- 是否启用菜单背景模糊
- 是否显示统计信息
- 是否启用音效反馈
- 总切换次数

可在 ModMenu 的模组配置界面或游戏内 H 菜单的 **设置** 按钮中修改。

## 创意功能

### 音效反馈
- 开启模块时播放高音提示音
- 关闭模块时播放低音提示音
- 可在设置中关闭音效

### 会话统计
菜单底部显示实时统计信息：
- 当前激活模块数/总模块数
- 总切换次数（跨会话累积）
- 当前会话时长

## ⚠️ 重要说明
**这只是一个娱乐模组，所有功能都是假的！**
- 不会真的飞行
- 不会真的透视
- 不会真的加速
- 所有功能都只是显示提示文字，没有实际效果
- 不会被反作弊检测（因为什么都没改）

## 构建方法

### 环境要求
- JDK 21
- 网络连接（首次构建需要下载依赖）

### 构建特定版本
1. 修改 `gradle.properties` 中的 `minecraft_version`
2. 运行构建命令：
   ```
   gradlew.bat build
   ```

### 构建所有版本
```
gradlew.bat buildAllVersions
```

### 国内用户加速
如果下载依赖慢，可以在 `~/.gradle/init.gradle` 中添加阿里云镜像：
```groovy
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/public' }
        maven { url 'https://maven.aliyun.com/repository/gradle-plugin' }
    }
}
```

## 安装方法
1. 安装对应版本的 Minecraft 和 Fabric Loader
2. 安装对应版本的 Fabric API
3. 将 `mrkun-bypass-1.2.0-mc{版本号}.jar` 放入 `.minecraft/mods` 文件夹
4. 启动游戏即可
